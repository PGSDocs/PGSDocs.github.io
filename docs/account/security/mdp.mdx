# Modification de Mot de Passe
Pour protéger votre compte Pro Gestion Soft, il est recommandé de mettre à jour régulièrement votre mot de passe. Cette section vous guidera pas à pas pour changer votre mot de passe en toute sécurité.

Pourquoi Modifier Votre Mot de Passe ?
La sécurité de votre compte repose en grande partie sur la robustesse de votre mot de passe. En changeant votre mot de passe régulièrement, vous diminuez les risques de piratage et gardez vos informations sensibles à l'abri. Voici quelques raisons de modifier votre mot de passe :

Renforcer la Sécurité : Un mot de passe régulièrement mis à jour rend votre compte moins vulnérable aux tentatives d'accès non autorisé.
Changer un Mot de Passe Faible : Si votre mot de passe actuel ne respecte pas les critères de sécurité (longueur, complexité, variété des caractères), il est recommandé de le remplacer par un mot de passe plus solide.
En Cas de Compromission : Si vous avez l’impression que votre compte a été compromis ou si vous avez réutilisé ce mot de passe ailleurs, il est crucial de le changer immédiatement.
Étapes pour Modifier le Mot de Passe
Suivez les étapes suivantes pour modifier votre mot de passe en toute sécurité :

Accéder à la Section Sécurité

Connectez-vous à votre compte Pro Gestion Soft.
Cliquez sur votre icône de profil dans le coin supérieur droit de l’écran, puis accédez à votre Espace Profil.
Dans le menu de votre Espace Profil, sélectionnez l’option Sécurité.
Renseigner les Informations de Sécurité

Dans la section Paramètres de Mot de Passe, saisissez votre mot de passe actuel pour confirmer votre identité.
Ensuite, entrez le nouveau mot de passe que vous souhaitez utiliser. Il doit respecter les critères de sécurité recommandés :
Longueur : Au moins 8 caractères.
Complexité : Comprend au moins une majuscule, une minuscule, un chiffre, et un symbole.
Saisissez à nouveau le nouveau mot de passe pour confirmer.
Valider le Changement

Après avoir rempli tous les champs requis, cliquez sur Enregistrer pour valider les modifications.
Vous recevrez une notification ou un e-mail confirmant que le changement de mot de passe a été effectué avec succès.
Critères pour un Mot de Passe Sûr
Pour garantir la sécurité de votre compte, voici quelques conseils pour choisir un mot de passe robuste :

Utilisez des mots de passe longs : Plus votre mot de passe est long, plus il est difficile à deviner.
Incluez des caractères variés : Mélangez majuscules, minuscules, chiffres, et symboles pour renforcer la sécurité.
Évitez les informations personnelles : N’utilisez pas des éléments faciles à deviner, comme votre nom, date de naissance, ou des combinaisons courantes (ex : "123456").
Changez votre mot de passe régulièrement : Cela réduit les risques de piratage, surtout si vous soupçonnez une tentative d’accès non autorisé.
Astuce : Accéder Rapidemment à la Section Sécurité
Pour un accès rapide, vous pouvez rechercher directement la section Sécurité en utilisant la barre de recherche. Saisissez des termes tels que "mot de passe" ou "changer mot de passe" pour retrouver la section souhaitée sans naviguer dans tout le menu.

En Cas d’Oubli du Mot de Passe Actuel
Si vous avez oublié votre mot de passe actuel, vous ne pourrez pas directement modifier votre mot de passe depuis votre profil. Dans ce cas :

Cliquez sur “Mot de passe oublié ?” sur la page de connexion.
Suivez les instructions pour recevoir un lien de réinitialisation par e-mail.
Utilisez ce lien pour définir un nouveau mot de passe et accéder de nouveau à votre compte.
Sécurité Supplémentaire : Activer la Double Authentification
Pour une sécurité optimale, pensez à activer la double authentification (2FA) dans la section Sécurité. Cela ajoutera une étape de vérification en plus de votre mot de passe, augmentant ainsi la protection de votre compte.

Bonnes Pratiques Après le Changement de Mot de Passe
Après avoir modifié votre mot de passe, suivez ces recommandations pour garantir une sécurité maximale :

Déconnectez-vous des Autres Sessions : Assurez-vous de déconnecter toutes les autres sessions actives pour prévenir tout accès non autorisé.
Ne Réutilisez pas l’Ancien Mot de Passe : Évitez de réutiliser le même mot de passe, même sur d’autres comptes.
Gardez une Trace Sécurisée : Si vous utilisez un gestionnaire de mots de passe, enregistrez-y le nouveau mot de passe pour éviter de l’oublier.
